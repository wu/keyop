Search Syntax

  • term1 term2 - Find documents with both terms (AND)
  • term1 OR term2 - Find documents with either term
  • "exact phrase" - Search for exact phrases
  • term1 -term2 - Include term1 but exclude term2
  • +term1 term2 - term1 is required, term2 is optional
  • term* - Wildcard search (e.g., test* matches test, testing, tested)
